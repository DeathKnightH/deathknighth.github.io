        # 工厂方法

        # 1. 概念

        工厂方法（Factory Method）又称虚拟构造函数，是一种创建型模式。

        # 2. 意图

        在父类中提供一个创建对象的接口，由子类决定创建什么类型的对象，即把实例化推迟到子类进行。
