    # 排序算法

    排序算法用于解决一类排序问题：

    输入：一个 n 个数的序列<a1, a2, a3, ... , an>，这个序列是一个线性的数据结构，通常是一个 n 元数组。

    输出： 输入序列的一个重排序<b1, b2, b3, ... , bn>，满足 b1 <= b2 <= b3 <= ... <= bn。
